# Latex

Latex 是一个文字排版工具，使用严格的代码匹配生成文档，对于编写论文，尤其是需要插入公式时非常方便，可以避免word中各种莫名其妙的排版错误。

## 使用指南

使用overleaf在线平台对文件进行解析，可以直接生成pdf格式，但需要注意引入中文字体支持：

1. 首先选择 xelatex 格式，默认的格式不支持中文
2. 添加中文包解析

```latex
\usepackage[UTF8]{ctex} % 这里加载 ctex 宏包
```
