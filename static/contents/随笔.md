# 个人思考

满打满算已经度过了人生的第29个年头，马上就临近孔子所谓的而立之年了。立的意思是说扎根在一个方向就不再改变了，不仅是在志向、工作等方面，也是在生活上。在往日的岁月里，我像一片随风飘摇的树叶，风到了哪里就把我吹到哪里，那我有着怎样的经历呢？和大多数中国人一样，不外乎在懵懂时玩耍（6年），而后是漫长的求学（16年），紧接着步入工作（6年）。往日的经历我其实都记不太清了，写这篇随笔就是为了回忆过往的我，以便指引我接下来的人生。

我小的时候出生在辽宁朝阳的一个偏远小山村，那时的村子里总是其乐融融的，虽然过着贫困的日子，但是我的乐园。总是说穷，其实到了现在我都不太能共情当时的自己了，只说说我还记得的几件小事吧：

- 每当下雨的时候我总是不太开心，因为不能出去玩耍，因为家里仅有两把雨伞，其中一把的骨架已经碎了一半，只有一半能撑开，一半则耷拉着；另一把则显得更为破旧，因为连把手都折断了。不知是不是因为类似的情景出现在我记忆中太多，我天然就学会了父母的节俭，我甚至更进一步，已经是非常抠抠搜搜了，夏天买雪糕时同学总是买5毛一根的，而我则是买5毛三根的，然后分到三天吃，那时的我真是个小机灵。

- 其他记忆较为清晰的事情就是娱乐活动了，比如用废旧的轮胎剪成皮筋，可以用来跳高、我甚至还会侧翻跟头，哈哈；用圆润的石子或者动物骨头（嘎拉哈）玩欻石子或者老虎出绵羊；还有用纸编成的四角（我那叫piaji）;玻璃球也是不错的玩具，但我都是从别人那里赢的，我还是挺厉害的。还有用半块砖磨成可以滚动的圆柱，用胳膊远远抛向远方的目标，类似现在的保龄球。还有用纸折的抓蜜蜂的工具，没少被蜜蜂蛰。虽然玩具上不够先进，但那时的我确实比现在的孩童快乐的多。

- 生活上也比较充实，那时家里有一头驴、两只羊，驴是干庄稼活的主力，而羊则是用来卖钱的。那时候种地唯一的营生，但是只够节衣缩食的生活，还要义务出苦力去大会战，比如挖树坑、修路等等，这种条件下牲畜的吃食就是很大的问题，所以我会跟着父母出去刌树杈子或者割草，我就这样长大了。

- 而后开始上学了，那时没有幼儿园，只有一个和蔼的先生在村里开私塾，不知道收学费没有，还是收些礼品之类的，细节我就没印象了。就是一群5、6岁的孩子一起在私塾里玩耍，也是因为家里人都比较忙，找个人带孩子罢了。但我母亲给我讲第一天去私塾的笑话，她前脚把我送到私塾，我后脚就偷摸跟着跑回家了，私塾先生发现我没在，等我到家时才追上我，路上约莫有半个小时的脚程。

- 私塾我只上了半年多，因为我母亲和学校的班主任是同学，我就选择在那时上小学了，有老师照顾确实带来了不少方便。班里一共有30多个同学，我和另外三个同学一直是前四名，但也不只学习这一条路。
